# Projeto-Detec-o-de-Fraudes---Curso-DIO
Codigo do projeto do curso de analise e automação com Python - DIO
#O projeto tem como objetivo analisar as fraudes em transações bancárias e para isso ussa um dataset publico para testes
#O código testa modelos de classificação para conseguir checar no melhor modelo, onde o Recall é o menor entre eles.
#Os modelos são: Classificação Logística, XGBoost, RandomForest e o Shap para a melhor combinação e modelo.
#Além de analisar a importancia e pesos das variaveis para os resultados dos modelos
